## epm-server

### start
virtual environment
```
python3 -m venv venv
source venv/bin/activate
```

django and djangoframework
```
pip3 install -r requirements/develop.txt
```

start django project
```
django-admin startproject epm .  #  be careful at the end of char '.'
cd epm
django-admin startapp epm
cd ..
```
